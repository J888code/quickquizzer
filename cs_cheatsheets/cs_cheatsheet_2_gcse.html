Object Oriented Programming AS

# Class definition
class Student:
    def __init__(self, name, year):
        self.name = name
        self.year = year
        self.score = 0

    def add_score(self, points):
        self.score += points

    def __str__(self):
        return f"{self.name} (Y{self.year}): {self.score}"

# Create objects
alex = Student("Alex", 10)
alex.add_score(95)
print(alex)  # Alex (Y10): 95
  • Class — blueprint for objects
  • Object — instance of a class
  • Attribute — variable belonging to object (self.name)
  • Method — function belonging to object
  • Constructor — __init__ sets up the object
  • Encapsulation — keeping data inside the class
  • Inheritance — child class gets parent's methods

Recursion AS

# Function that calls itself
def factorial(n):
    if n <= 1:          # base case!
        return 1
    return n * factorial(n-1)

factorial(5)
# = 5 * factorial(4)
# = 5 * 4 * factorial(3)
# = 5 * 4 * 3 * 2 * 1
# = 120

# Fibonacci
def fib(n):
    if n <= 1: return n
    return fib(n-1) + fib(n-2)
Exam tip: Always need a BASE CASE — otherwise infinite recursion = crash!

Images, Sound & Compression

Images

  • Pixel — single dot of colour in an image
  • Resolution — number of pixels (width × height)
  • Colour depth — bits per pixel. 24-bit = 16.7 million colours
  • File size = pixels × colour depth (bits) / 8 (bytes)

Sound

  • Sample rate — how many times per second sound is measured (Hz)
  • Bit depth — bits per sample, affects quality
  • File size = sample rate × bit depth × duration × channels

Compression

  • Lossy — removes data permanently. Smaller file, lower quality (MP3, JPEG)
  • Lossless — no data lost. Can restore exactly. Larger file (PNG, FLAC)
  • Run-Length Encoding (RLE) — replaces repeated data with count+value

Software & Operating Systems

  • OS — manages hardware, memory, files, user interface

  • Device Driver — software allowing OS to communicate with hardware
  • Virtual Memory — using storage as overflow RAM when RAM is full
  • Multitasking — appearing to run multiple programs simultaneously

Searching Algorithms

Binary Search (much faster than linear!)

def binary_search(lst, target):
    low, high = 0, len(lst) - 1
    while low <= high:
        mid = (low + high) // 2
        if lst[mid] == target:
            return mid
        elif lst[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1
List MUST be sorted for binary search to work!

Complexity Comparison

AlgorithmTime Complexity
Linear searchO(n) — check every item
Binary searchO(log n) — halve each time

Sorting Algorithms

Merge Sort (faster, O(n log n))

def merge_sort(lst):
    if len(lst) <= 1:
        return lst
    mid = len(lst) // 2
    left = merge_sort(lst[:mid])
    right = merge_sort(lst[mid:])
    return merge(left, right)

Cybersecurity In Depth

Types of Attack

  • DoS/DDoS — flood server with requests to crash it
  • Man in the Middle — intercept communications between two parties
  • SQL Injection — inject malicious SQL into input fields
  • XSS — inject malicious scripts into websites
  • Keylogging — record every keypress to steal passwords

Defences

  • Input validation — check data before processing
  • Parameterised queries — prevent SQL injection
  • HTTPS/TLS — encrypt data in transit
  • Hashing passwords — never store plain text passwords
  • Penetration testing — ethically hack your own systems to find weaknesses
Pen testing is literally the ethical hacking career path — this is your area!
