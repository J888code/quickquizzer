For Loops

# Loop through list
for item in ["a", "b", "c"]:
    print(item)

# Loop with index
for i in range(5):
    print(i)  # 0,1,2,3,4

# range(start, stop, step)
for i in range(0, 10, 2):
    print(i)  # 0,2,4,6,8

# enumerate gives index + value
for i, val in enumerate(["a","b","c"]):
    print(i, val)
enumerate() is cleaner than range(len()) — learn it!

File Handling ⚠️ NEW

# Write to a file
with open("scores.txt", "w") as f:
    f.write("Alex: 95\n")
    f.write("Sam: 87\n")

# Read from a file
with open("scores.txt", "r") as f:
    content = f.read()
    print(content)

# Read line by line
with open("scores.txt", "r") as f:
    for line in f:
        print(line.strip())
Use "a" instead of "w" to append to a file without overwriting.

How Computers Work

  • CPU — Central Processing Unit. The brain. Fetches, decodes, executes instructions
  • RAM — Random Access Memory. Temporary storage while running. Fast but loses data when off
  • ROM — Read Only Memory. Permanent, stores boot instructions. Doesn't change
  • Storage (HDD/SSD) — Permanent storage for files. Slower than RAM
  • ALU — Arithmetic Logic Unit. Part of CPU that does maths and logic
  • Control Unit — Part of CPU that manages instruction flow
  • Cache — Tiny ultra-fast memory inside CPU for frequently used data
  • Clock Speed — How many instructions per second (GHz). Higher = faster
⚡

Fetch-Decode-Execute Cycle

  • Fetch — CPU gets next instruction from RAM using the Program Counter (PC)
  • Decode — Control Unit works out what the instruction means
  • Execute — ALU carries out the instruction
  • Repeat — billions of times per second
This cycle is the foundation of how every program you've ever written actually runs.

Key Registers

  • PC — Program Counter: address of next instruction
  • MAR — Memory Address Register: address being accessed
  • MDR — Memory Data Register: data being transferred
  • ACC — Accumulator: stores result of ALU operation
🌐

Networks Basics

  • LAN — Local Area Network. Small area (school, home)
  • WAN — Wide Area Network. Large area (internet is the biggest WAN)
  • IP Address — Unique address for each device on a network
  • MAC Address — Hardware address built into network card. Never changes
  • Router — Directs data between networks
  • Switch — Connects devices within a LAN
  • DNS — Domain Name System. Converts google.com to an IP address
  • HTTP/HTTPS — Protocol for web pages. S = encrypted
  • Packet — Small chunk of data. All data travels in packets
🔒

Cybersecurity Basics

  • Malware — Malicious software. Includes viruses, worms, ransomware, spyware
  • Phishing — Fake emails/sites to steal passwords or data
  • Brute Force — Trying every possible password combination
  • SQL Injection — Inserting malicious code into database queries
  • Social Engineering — Manipulating humans rather than hacking systems
  • Encryption — Scrambling data so only authorised users can read it
  • Firewall — Monitors and filters network traffic
  • Two-Factor Auth — Password + second verification (code, fingerprint)

Algorithms

Linear Search

def linear_search(lst, target):
    for i, val in enumerate(lst):
        if val == target:
            return i
    return -1  # not found

Bubble Sort

def bubble_sort(lst):
    n = len(lst)
    for i in range(n):
        for j in range(0, n-i-1):
            if lst[j] > lst[j+1]:
                lst[j], lst[j+1] = lst[j+1], lst[j]
    return lst
Know how to trace through these by hand — exam questions often ask you to show steps.
